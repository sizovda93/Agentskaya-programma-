Проблема

Для бизнеса

  • Агентская сеть растёт, но управление ведётся вручную
  • Нет единой системы учёта лидов и выплат
  • Споры «чей клиент» решаются в чатах
  • Нет контроля качества и обучения партнёров
  • Менеджер тратит 80% времени на рутину

Для агентов

  • Непонятно, сколько заработал и за что
  • Нет инструментов для привлечения клиентов
  • Нет обучения и адаптации
  • Нет прозрачности выплат
  • Нет мотивации расти

Решение

Агентская программа — веб-платформа, которая автоматизирует управление партнёрской сетью: от регистрации агента до выплаты комиссии. Вся цепочка — лиды, обучение, контроль качества, финансы, коммуникации — в одном месте.

Возможности платформы

12 подсистем, построенных за 9 этапов

Ядро платформы

Управление лидами

  • Создание лидов агентом / менеджером
  • 7-этапная воронка: new → won/lost
  • Автоматическое обнаружение дублей по телефону/email
  • Conflict resolution flow (4 варианта)
  • Ownership policy: первый создавший = владелец
  • Timeline событий по каждому лиду

Финансы и выплаты

  • Автоматическое создание payout при won
  • State machine: pending → processing → paid
  • 3 уровня партнёров: Base (25%), Silver (30%), Gold (35%)
  • Snapshot расчёта: база, ставка, tier, бонус
  • Auto-upgrade Base → Silver при 5 won
  • Rejection с обязательной причиной
